# Oded Nachshon - Portfolio

A modern, distinctive portfolio website showcasing the career and expertise of Oded Nachshon, Head of Wix Editor & Media R&D.

## Features

- **Bold Design**: Cyberpunk-inspired aesthetic with Space Mono & Crimson Pro typography
- **Responsive**: Mobile-optimized with smooth animations
- **Performance**: Pure HTML/CSS/JS - no frameworks, fast loading
- **Interactive**: Scroll reveals, parallax effects, and smooth navigation
- **Professional**: Showcases experience, skills, education, and contact information

## Live Site

Visit the portfolio at: [https://itayshmool.github.io/oded-nachshon-portfolio/](https://itayshmool.github.io/oded-nachshon-portfolio/)

## Local Development

Simply open `index.html` in your browser. No build step required.

## Deployment

Deployed using GitHub Pages from the `main` branch.

## Design Highlights

- **Typography**: Space Mono (monospace) paired with Crimson Pro (serif)
- **Color Palette**: Dark theme with cyan, magenta, and yellow accents
- **Effects**: Grid background, noise texture, gradient glows, animated reveals
- **Layout**: Asymmetric composition with diagonal sections and offset elements

## Technologies

- Pure HTML5
- CSS3 (Grid, Flexbox, Animations, Custom Properties)
- Vanilla JavaScript (Intersection Observer, Scroll Events)
- Google Fonts API
